Проект 2

Калькулятор землекопов

Интерактивное веб-приложение для расчета рабочих

← Предыдущий проект Следующий проект →

О проекте

Интерактивное веб-приложение для расчета необходимого количества землекопов для трех канав. Пользователь может выбрать тип бригады (механизированная или обычная) и указать длину каждой канавы. Приложение динамически рассчитывает количество рабочих и предоставляет интерактивные результаты.

Ключевые функции

Динамический интерфейс

Генерация элементов интерфейса через JavaScript

Интерактивный выбор

Визуальный выбор бригад с подсветкой выбранного варианта

Алгоритм расчета

Автоматический расчет рабочих на основе типа бригады и длины канавы

Пошаговые подтверждения

Интерактивный процесс подтверждения результатов для каждой канавы

Визуальная обратная связь

Цветовое кодирование статусов канав (работает/отдыхает)

Итоговая статистика

Подробный отчет с общей статистикой по всем канавам

Техническая реализация

Проект демонстрирует работу с DOM-элементами, обработку пользовательского ввода, создание динамического контента и использование функций высшего порядка. Особое внимание уделено пользовательскому опыту через интерактивные подтверждения и визуальную обратную связь.

Используемые технологии

HTML5 CSS3 JavaScript ES6+

Как использовать

  1. Укажите длину каждой канавы в метрах
  2. Выберите тип бригады для каждой канавы
  3. Нажмите "Рассчитать количество землекопов"
  4. Подтверждайте результаты для каждой канавы через диалоговые окна
  5. Просмотрите итоговую статистику
